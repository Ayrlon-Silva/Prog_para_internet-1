WeatherAPI

Descrição

A WeatherAPI fornece dados climáticos atuais, previsão e histórico. É necessário obter uma API key gratuita no site weatherapi.com.

Demonstrativo via Curl

Comando:

curl "http://api.weatherapi.com/v1/current.json?key=SUA_CHAVE&q=São Paulo&lang=pt"

Exemplo de saída esperada:

{
  "location": {
    "name": "São Paulo",
    "country": "Brazil"
  },
  "current": {
    "temp_c": 23.0,
    "condition": {
      "text": "Parcialmente nublado",
      "icon": "//cdn.weatherapi.com/..."
    }
  }
}
