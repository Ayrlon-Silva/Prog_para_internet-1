Agify API

Descrição

A Agify API estima a idade de uma pessoa com base no nome fornecido. Não requer autenticação.

Demonstrativo via Curl

Comando:

curl "https://api.agify.io?name=maria"

Exemplo de saída esperada:

{
  "name": "maria",
  "age": 34,
  "count": 53492
}
