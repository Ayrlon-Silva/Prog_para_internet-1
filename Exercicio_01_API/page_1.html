OMDb API

Descrição

A OMDb API (Open Movie Database) é uma interface REST que fornece dados completos sobre filmes, séries e episódios de TV. Ela permite buscar informações como título, ano, duração, sinopse, pôster e mais.

Demosntração via Curl

Comando:

curl "http://www.omdbapi.com/?t=TITULO_DA_OBRA&apikey=SUA_APIKEY"

Exemplo de saída esperada:


        {
        "Title": "Spider-Man",
        "Year": "2002",
        "Rated": "PG-13",
        "Released": "03 May 2002",
        "Runtime": "121 min",
        ...
        "Director": "Sam Raimi",
        "Poster": "img.jpg",
        "Type": "movie",
        "Response": "True"
        }
